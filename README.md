# Random Number & Image Display (TestFront)

live at https://music10019-arch.github.io/testFront.github/

A simple interactive web page with two features:
- Click a button to display a random number (1-5)
- Hover over a button to display a random image from GitHub

## Features

### Left Block: Random Number Generator
- Generates a random number between 1 and 5
- Updates on each button click

### Right Block: Random Image Display
- Fetches images from a GitHub repository
- Displays a random image on mouse hover
- Includes loading indicator and fade-in effect
- Caches image URLs for better performance

## Technologies Used

- HTML5
- CSS3 (Flexbox layout)
- Vanilla JavaScript (ES6+)
- GitHub API

## Usage

Simply open `index.html` in your web browser. No build process or dependencies required.

## API Source

Images are fetched from: `https://github.com/music10019-arch/chrisHHH.github/tree/main/images`
